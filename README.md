## Ricky and Morty APP

### Features:

---

##### Here's features included in this project

- Character and thier Deatils Page
- Pagination
- Awesome modern minimalist UI

### Technologies Used:

---

1. React.js
2. ReactStrap
3. CSS
4. Node.js 

### How to start?

---

##### To start the project along with me just download the start-up file and run available script

```javascript
// Client Side
// first install all necessary dependencies

npm i

// next run

npm start

// server Side

npm init

// next run

nodemon ./index.js

```
